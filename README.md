# Sistema de Gerênciamento de Livros - BookHub

BookHub é um aplicativo desktop desenvolvido em javaFX que tem o intuito de permitir que seus usuários possam gerênciar e catalogar seus livros de forma organizada.

Com ele, é possível criar perfis, manter listas de leitura (como *Lendo*, *Favoritos*, *Quero Ler* e *Lidos*), pesquisar títulos e personalizar o visual do perfil (Foto de Perfil e Imagem de Fundo).
