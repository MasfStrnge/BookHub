# Sistema de Gerênciamento de Livros - BookHub

O BookHub é um sistema de gerenciamento de livros desenvolvido como uma aplicação desktop Java, onde o usuário pode catalogar e organizar sua coleção de livros pessoal de forma intuitiva e personalizada, possuindo funcionalidades completas para controle de leitura e organização da coleção.
Um usuário é capaz de buscar por livros através de filtros baseados em título, autor, gênero e ISBN, visualizar as informações detalhadas de cada obra e adicioná-las a listas predefinidas conforme seu status de leitura: Favorito, Lendo, Lido ou Quero Ler.  o sistema permite a criação de listas customizadas, onde o usuário pode catalogar seus livros por critérios próprios, como temática ou qualquer outra categoria que desejar. Além disso, o usuário pode avaliar livros marcados com o status de Favorito ou Lido, personalizar seu perfil com foto e imagem de fundo.
